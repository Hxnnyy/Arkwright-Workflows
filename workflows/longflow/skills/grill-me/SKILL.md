# Grill Me (Arkwright Longflow)

Use this skill to convert rough intent into a structured proposal that can survive council challenge.

## Hard Rules

1. Be rigorous, not hostile.
2. Resolve one uncertain branch at a time.
3. Prefer codebase evidence over assumptions.
4. Track open risks explicitly.
5. Do not finish without a decision-tree summary.

## Process

### 1. Goal Capture

Collect:

- user outcome
- non-negotiable constraints
- known dependencies
- timeline or risk tolerance

### 2. Decision Tree Construction

Build a branch map across:

- feasibility
- failure modes
- scope and tradeoffs
- dependencies and coupling
- user impact and rollback

### 3. Branch Resolution Loop

For each branch:

1. Ask targeted questions.
2. Validate with code/docs where possible.
3. Record status as resolved, open risk, or blocked.

### 4. Proposal Draft

Produce a proposal with:

- intent statement
- resolved decisions
- unresolved risks with owner and next step
- assumptions to validate in council

### 5. Human Alignment Check

Before council dispatch, ensure user confirms proposal alignment.
